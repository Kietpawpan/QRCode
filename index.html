Work Standard

MNRE-SLC-003

(สำเนา)

ประกาศสำนักงานสิ่งแวดล้อมและควบคุุมมลพิษที่ ๑๐
เรื่อง กำหนดมาตรฐานการจัดทำ QR Code

เพื่อให้การจัดทำและใช้งาน QR Code มีความสะดวก ปลอดภัย ไม่มีการโฆษณา สำนักงานสิ่งแวดล้อมและควบคุุมมลพิษที่ ๑๐ จึงกำหนดมาตรฐานการจัดทำ QR Code โดยขอให้เจ้าหน้าที่ในสังกัดสำนักงานสิ่งแวดล้อมและควบคุุมมลพิษที่ ๑๐ ใช้โปรแกรมตามท้ายประกาศนี้ในการจัดทำ QR Code ความกว้าง ๑๕๐px ความสูง ๑๕๐px นามสกุล png และหากแสดง QR Code ไว้ท้ายหนังสือ ให้ปรับขนาดกว้างยาวเป็น ๑.๕ เซนติเมตร พร้อมระบุชื่อของ QR Code นั้น หรือชื่อบริการปลายทาง ไว้ด้านล่าง ด้วยตัวอักษรขนาดปกติ


ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป


ประกาศ ณ วันที่ ๙ กรกฎาคม พ.ศ. ๒๕๖๘

(ลงนาม) มนตรี เกียรติเผ่าพันธ์
(นายมนตรี เกียรติเผ่าพันธ์)
