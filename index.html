ที่ ๓๔ /๒๕๖๘
เรื่อง  มาตรฐานการจัดทำ QR Code

---------------------------

เพื่อให้ QR Code ที่จัดทำโดยสำนักงานสิ่งแวดล้อมและควบคุมมลพิษที่ ๑๐ (ขอนแก่น) มีความมั่นคงปลอดภัยทางไซเบอร์ สร้างความน่าเชื่อถือ และเสริมสร้างภาพลักษณ์ที่ดีแก่องค์กรภาครัฐในยุคดิจิทัล

อาศัยอำนาจตามมาตรา ๓๓ วรรคสอง แห่งพระราชบัญญัติระเบียบบริหารราชการแผ่นดิน พ.ศ. ๒๕๓๕ และคำสั่งกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม ที่ ๒๐๔/๒๕๖๘ ลงวันที่ ๑๗ มิถุนายน พ.ศ. ๒๕๖๘ จึงให้เจ้าหน้าที่ในสังกัดสำนักงานสิ่งแวดล้อมและควบคุมมลพิษที่ ๑๐ (ขอนแก่น) ใช้โปรแกรมท้ายคำสั่ง ในการจัดทำ QR Code โดยกำหนดความกว้างและความสูง ขนาด ๑๕๐ พิกเซล เมื่อวางแนบท้ายหนังสือราชการ ปรับขนาด QR Code ให้มีความกว้างและความสูง เท่ากับ ๑.๕ เซนติเมตร พร้อมทั้งระบุชื่อของ QR Code เป็นชื่อย่อของเอกสารแนบหรือสิ่งที่ส่งมาด้วย แล้วแต่กรณี ไว้ด้านล่างของ QR Code นั้น และใช้ฟอนต์ปกติ ขนาด ๑๔ พิกเซล หรือขนาดที่เหมาะสม
