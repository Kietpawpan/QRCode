Work Standard

MNRE-SLC-003

สำเนา

ประกาศส่วนข้อมูลข่าวสารและบริการร่วม
เรื่อง กำหนดมาตรฐานการจัดทำ QR Code

เพื่อให้การใช้ QR Code มีความสะดวกและปลอดภัย ส่วนข้อมูลข่าวสารและบริการร่วม จึงกำหนดมาตรฐานการจัดทำ QR Code โดยขอให้เจ้าหน้าที่ในสังกัดส่วนข้อมูลข่าวสารและบริการร่วม ใช้โปรแกรมตามท้ายประกาศนี้ในการจัดทำ QR Code ความกว้าง 150px ความสูง 150px นามสกุล png ปรับเปลี่ยนชื่อไฟล์ได้ และหากแสดงท้ายหนังสือ ให้ปรับขนาดกว้างยาวเป็น 1.5 เซนติเมตร พร้อมระบุชื่อของ QR Code นั้น หรือชื่อบริการปลายทาง ไว้ด้านล่าง ด้วยตัวอักษรขนาดปกติ


ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป


สั่ง ณ วันที่ 12 กันยายน 2566

(ลงนาม) มนตรี เกียรติเผ่าพันธ์
(นายมนตรี เกียรติเผ่าพันธ์)
