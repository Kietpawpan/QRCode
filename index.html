Work Standard

EPO-10-2501


ประกาศสำนักงานสิ่งแวดล้อมและควบคุมมลพิษที่ ๑๐ (ขอนแก่น)
เรื่อง มาตรฐานการจัดทำ QR Code

เพื่อให้การจัดทำและการใช้งาน QR Code ของสำนักงานสิ่งแวดล้อมและควบคุมมลพิษที่ ๑๐ (ขอนแก่น) มีความสะดวก ปลอดภัย ไม่มีโฆษณาคั่น และป้องกันการละเมิดข้อมูลส่วนบุคคลของผู้ใช้งาน

อาศัยอำนาจตามมาตรา ๓๓ วรรคสอง แห่งพระราชบัญญัติระเบียบบริหารราชการแผ่นดิน พ.ศ. ๒๕๓๕ และคำสั่งกระทรวงทรัพยากรธรรมชาติและสิ่งแวดล้อม ที่ ๒๐๔/๒๕๖๘ ลงวันที่ ๑๗ มิถุนายน พ.ศ. ๒๕๖๘ จึงกำหนดมาตรฐานการจัดทำ QR Code สำหรับแนบท้ายหนังสือสำนักงานสิ่งแวดล้อมและควบคุุมมลพิษที่ ๑๐ (ขอนแก่น) โดยขอให้เจ้าหน้าที่ในสังกัดสำนักงานสิ่งแวดล้อมและควบคุุมมลพิษที่ ๑๐ (ขอนแก่น) ใช้โปรแกรมตามท้ายประกาศนี้ ในการจัดทำ QR Code ความกว้าง ๑๕๐px ความสูง ๑๕๐px นามสกุล png ทั้งนี้ เมื่อใช้แนบท้ายหนังสือราชการ ขอให้ปรับขนาดความกว้างและความสูง เป็น ๑.๕ เซนติเมตร พร้อมระบุชื่อของ QR Code (ชื่อบริการปลายทาง) ไว้ด้านล่าง ด้วยตัวอักษรปกติ ขนาดตัวอักษร ๑๔px


ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป


ประกาศ ณ วันที่ ๑ สิงหาคม พ.ศ. ๒๕๖๘

(ลงนาม) มนตรี เกียรติเผ่าพันธ์
(นายมนตรี เกียรติเผ่าพันธ์)
ผู้อำนวยการสำนักงานสิ่งแวดล้อมและควบคุมมลพิษที่ ๑๐
