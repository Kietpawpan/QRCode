Work Standard

MNRE-SLC-004

สำเนา

ประกาศส่วนข้อมูลข่าวสารและบริการร่วม
เรื่อง กำหนดมาตรฐานการจัดทำ QR Code (ฉบับที่ 2)

เพื่อให้การจัดทำ QR Code มีความเรียบง่ายและปลอดภัย จึงกำหนดมาตรฐานการจัดทำ QR Code กรณี URL มีจำนวนตัวอักษรมาก ให้ใช้โปรแกรมตามท้ายประกาศนี้ในการจัดทำ Short URL ก่อน แล้วจึงจัดทำ QR Code จาก Short URL นั้น และให้มีขนาด 1.5 ✕ 1.5 เซนติเมตร


ทั้งนี้ ตั้งแต่บัดนี้เป็นต้นไป


สั่ง ณ วันที่ 13 กันยายน 2566

(ลงนาม) มนตรี เกียรติเผ่าพันธ์
(นายมนตรี เกียรติเผ่าพันธ์)
